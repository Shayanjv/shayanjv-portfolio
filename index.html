InvigilAI — Real-Time Exam Proctoring Platform

Dec 2024 – Feb 2026

A production-grade, real-time AI-powered online examination proctoring system designed for university internal assessments. Built as a three-tier Dockerized architecture — Student Portal (React + Vite), Admin Dashboard (React + TypeScript), and FastAPI Backend — with PostgreSQL, Redis, and MinIO, all orchestrated via Docker Compose.

  • Concurrent AI Detection Engine — 3 models run in parallel via ThreadPoolExecutor: YOLOv8 (phone & person detection), MediaPipe Face Mesh (eye tracking, head posture, mouth movement), and MediaPipe Hands — achieving 95%+ accuracy at <500ms latency with configurable sensitivity thresholds
  • Real-Time WebSocket Pipeline — JWT-authenticated WebSocket connections stream base64 video frames at 10–15 FPS; server decodes, runs AI inference, and pushes violation alerts back instantly with event-type classification
  • Dual Authentication & Security — Password + Face Recognition biometric login, JWT Bearer tokens, SEB (Safe Exam Browser) hash validation, CORS restriction, per-exam IP range enforcement, and role-based access control
  • Compliance Scoring & Auto-Grading — Weighted compliance algorithm with noise filtering (>10 occurrences threshold), face detection rate tracking, and automatic MCQ grading with per-question marks allocation
  • Admin Dashboard (12+ Components) — Live student monitoring table, evidence vault with MinIO screenshots, exam creator with MCQ/short-answer, AI sensitivity settings, real-time stats bar, and per-student compliance modal
  • LTI 1.3 LMS Integration — Hybrid deployment as Stand-alone or LMS Plugin for Moodle, Canvas & Blackboard with auto-provisioning of students and gradebook sync

MediMind — AI-Powered Medicine Management App

2025

A production-grade Android application that leverages AI-powered prescription scanning to automate medicine management. Built with Jetpack Compose and Clean Architecture (4-layer separation), the app uses a dual-AI verification pipeline to extract medicine details from prescriptions, auto-generate reminders with exact alarms, and track dose adherence — all synced to the cloud via Supabase with offline-first Room caching.

  • AI Prescription Scanner — Scan handwritten/printed prescriptions using CameraX; AI extracts medicine name, dosage, frequency, duration, quantity, side effects, and instructions automatically
  • Dual-AI Verification — Two independent models (Mistral via NVIDIA API + Gemma/Gemini) cross-verify extracted data to catch dosage errors, quantity miscalculations, and missing warnings for clinical safety
  • Smart Auto-Scheduling — One-tap automation creates medicines, reminders, and exact alarms from scan results; alarms survive Doze mode, device restarts, and app kills via setExactAndAllowWhileIdle
  • Real-Time Adherence Tracking — Daily adherence percentage dashboard with dose status management (Taken, Missed, Snoozed, Pending) and automated missed-dose detection via background workers
  • Offline-First Sync — Room DB operates independently; bi-directional Supabase sync every 6 hours with conflict resolution (server-wins for medicines, local-wins for dose logs)
  • Firebase Auth & Security — Phone OTP + Google Sign-In via Firebase Authentication; Row Level Security (RLS) on all 4 Supabase tables ensures users can only access their own data
Kotlin Jetpack Compose Clean Architecture Hilt Room Mistral AI Gemma / Gemini CameraX Supabase Firebase WorkManager Material 3

Vastra — AI-Powered Wardrobe Manager

2025

A production-grade, cross-platform mobile application built for the Indian market that uses AI to transform wardrobe management. Built with Flutter and Clean Architecture (35 use cases across 9 modules), Vastra digitizes wardrobes, generates weather-aware outfit suggestions via Gemini API, tracks laundry cycles, and predicts fabric health — powered by a Node.js + PostgreSQL backend on Azure.

  • AI Outfit Recommendations — Gemini API generates personalized outfit suggestions based on weather (OpenWeather), occasion, skin tone, and style history with style scores and AI explanations
  • Clean Architecture at Scale — Strict 3-layer separation (Presentation → Domain → Data) with 35 use cases, 14 entities, 8 repository contracts, and 57 custom widgets — 0 lint issues across entire codebase
  • Offline-First Design — Isar local NoSQL database with seamless API sync via Dio interceptors and automatic token refresh; reliable on 2G/3G networks and 2 GB RAM devices
  • India-First Localization — Full Indian ethnic wear taxonomy (Saree, Kurta, Sherwani, Lehenga), regional festivals (Diwali, Eid, Pongal, Navratri), multi-day wedding planner, ₹ currency, and +91 phone format
  • Fabric Quality Prediction — Timeline-based degradation model with per-part health tracking (collar, sleeve, sole, lining) and predictive care recommendations with visual graphs
  • Freemium Monetization — Razorpay payment integration (₹99/mo or ₹799/yr) with Firebase Auth (OTP + Google Sign-In), FCM push notifications, Crashlytics, and Analytics
