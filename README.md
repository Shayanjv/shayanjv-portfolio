# Shayana J V — Developer Portfolio 🚀

A modern, high-performance developer portfolio showcasing full-stack projects and technical skills. Built with **Vite**, **Vanilla JS/CSS**, and a custom **Glassmorphism** design system.

![Portfolio Preview](https://via.placeholder.com/800x400?text=Portfolio+Preview)

## ✨ Features

- **🎨 Dark Theme Excellence:** Custom gradients with purple/cyan accents.
- **💎 Glassmorphism UI:** Frosted glass effect cards and overlays.
- **⚡ High Performance:** Built on Vite for instant HMR and optimized builds.
- **📱 Fully Responsive:** Adaptive layout for mobile, tablet, and desktop.
- **⌨️ Interactive Elements:** Typing animations, scroll reveals, and smooth navigation.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Custom Properties), Vanilla JavaScript
- **Build Tool:** Vite
- **Design:** Custom CSS, FontAwesome Icons
- **Deployment:** Optimized for Vercel

## 🚀 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Shayanjv/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
